# Firmware

## Installation

Pull this `git` and Run this as a root:
```sh
bash ~/git/Firmware/util/install_dependencies.sh
```

In case of Error (with `shim-signed`):

```sh
sudo apt-get install shim-signed
```

Set teensy :
```sh
sudo cp ~/git/teensy/49-teensy.rules /etc/udev/rules.d/
```
