# Firmware


## Edit Keymap

```sh
gedit /Firmware/qmk/keyboards/ergodox/ez/dawkiny/keymap.c
cd /Firmware/qmk/keyboards/ergodox/
make ez dawkiny

cd /Firmware
./teensy/teensy.64bit &
```
